Contraception Usage by Education Level in Select Low-Income Countries

The data in this table shows the percentage of contraception usage in 75 low-income countries and the under-5 mortality rate (U5MR) in each country. The blue columns show the % contraception use for populations with no education, primary education only, and secondary education or greater, as well as the averaged usage. The red column shows the U5MR.

Darker colors indicate a higher percentage of usage or higher mortality rate. The data is by default sorted alphabetically by country name, but columns are sortable by clicking on the column header.

Sources: World Health Organization; World Bank. Only countries with complete data are represented.
